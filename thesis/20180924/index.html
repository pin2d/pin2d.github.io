電繪矯正班總集篇1 - 正確的骨架觀念
2018-09-25|教學影片
字數總計: 1,245|閱讀時長: 5 分鐘

大多數人之所以骨架會畫不好，並非是對人體的不理解，而是沒掌握好畫骨架的原理，造成後續學不好，就好比學英文應該先學發音，而不是先學句型。所以本系列的教程，是以初學者為主，一步一步將人體每個環節的骨架畫法學起來，用於之後的人體練習才會更快速！

主講者：CORNLEE玉米力

喜歡的人可以去FB給他鼓勵一下喔~~
https://www.facebook.com/CORNLEE.YUMILEE/﻿

PART.1 - 頭部骨架三步驟 Steps of the Head

這次教學是關於頭部骨架，讓大家在畫頭部時不再擔心只會畫正面!用三步驟簡單畫出立體的頭型，除了介紹漫畫派與寫實派兩種的起手方式還告訴你該如何練習歐!
Tutorial on head figure this time.Don’t have to worry about only knowing how to draw the front view!Using 3 steps to draw the 3d head figure. Not only does it cover both manga and realistic style. It also tells you how to practice!

PART.2 - 簡單畫胸腔 Drawing the upper body

這次教學是關於胸腔骨架，透過簡單的方式幫助大家認識胸腔，再來教大家用四步驟簡單畫出男、女生的胸腔、骨盆!最後還會告訴你該如何練習歐!
This one is about structure of the rib cage. Using simple method to let us learn about it. Following it with 4 steps on how to draw male and female rib cage and pelvis. Lastly it also teaches how to practice!

PART.3 - 手臂好容易 Drawing Hands

這次教學是關於手臂骨架，我們會透過簡易的方式讓大家去理解手臂的比例跟位置，再來教大家用三步驟輕鬆畫出自然的手臂!!最後還會告訴你該如何練習歐!
This tutorial is about hand figure,We will use simple method to let everyone learn arm ratios and position, Then teach how to draw natural looking arms. Also includes how to practice!

PART.4 - 勇敢畫手掌 Drawing Hands

這次教學是關於手掌骨架，我們會用一些小技巧幫助大家理解手掌跟手指之間的結構跟位置，再來教大家用三步驟簡單畫出手掌!最後還會告訴你該如何練習歐!
This time it’s about hand structure. We’ll use some little tips to help everyone understand the structure and position of hand and fingers. Then we’ll teach everyone how to draw hands with 3 steps! Lastly, there will also be info on how to practice!

PART.5 - 美胸小技巧 Breast Tutorial

今天的重點是胸部的骨架草稿畫法，大家剛開始學習的時候畫到胸部通常會畫得不自然，所以這次會告訴大家胸部的生長規則還會提供讓身體線條吸睛的小技巧，今天就來好好學習胸部的畫法吧!
Today we’re focusing on starting out breast’s sketch. Usually everyone draw breasts that look unnatural. This time we’ll talk about breast’s form and how they grow
Also includes tips on how to make body curves more sexy. Let us learn about how to draw breasts today!

PART.6 - 下肢好穩固(上) Learning Lower Limbs A

今天的重點是下肢的骨架草稿畫法，大家剛開始學習的時候通常都是畫臉開始，頂多到胸部，而下肢的部分通常沒有很多研究。而要開始研究的時候又因為資料太多又太複雜而吸收不良，變成作品都只有上半身的狀況。
今天就來好好學習簡單的下肢骨架觀念吧～
Today’s topic is drawing, Most people start from learning to draw the head, at most to the chest. The bottle half usually comes last, but it can be hard to learn because the info sometimes are too overwhelming. So we end up with only the upper half. Let’s try to learn about lower limbs

PART.7 - 下肢好穩固(下) Learning Lower Limbs B

今天主要講腳掌的畫法，大家常常仔細地研究了腳掌肌肉和線條，花了很多力氣結果最後多半是穿了鞋子，實際肌肉並不常用到，比較好的做法是剛開始就是以畫鞋子為目的去設計骨架，不僅有利於創作，就算真的要腳掌肌肉時只要找好資料就可以了！
We’re talking about feet today. Somtimes people study feet very carefully, but they end up just drawing the shoes. The muscles usually don’t show, the better method is to just learn shoes for beginners. It’s more convenient, and refer to reference when you need to draw actual feet.
